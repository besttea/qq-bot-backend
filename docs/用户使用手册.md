# 用户使用手册 (User Manual)

**版本**: 1.0
**日期**: 2026-02-06

---

## 1. 快速入门

### 1.1 访问方式
*   **学生端**: 直接在 QQ 中与机器人对话，或访问移动端 Web 页面 `https://student.example.com`。
*   **教师端**: 访问 PC 端管理后台 `https://admin.example.com`。

### 1.2 账号绑定
1.  **添加机器人**: 在 QQ 群或私聊中添加机器人好友。
2.  **身份认证**: 发送指令 `/bind 姓名 学号`（例如 `/bind 张三 2023001`）。
3.  **成功反馈**: 机器人回复“绑定成功，欢迎加入！”即完成注册。

## 2. 教师操作指南

### 2.1 创建班级
1.  登录管理后台，点击左侧菜单 **“班级管理”**。
2.  点击右上角 **“新建班级”** 按钮。
3.  输入班级名称（如“2026春季-软件工程”），点击确认。
4.  系统将生成一个 **6位邀请码**，请将此码发送给学生。

### 2.2 发布作业
1.  进入指定班级，点击 **“作业列表”** -> **“发布作业”**。
2.  填写作业详情：
    *   **标题**: 简明扼要（如“第一次实验报告”）。
    *   **说明**: 详细要求或评分标准。
    *   **附件**: 可上传 PDF/PPT 课件。
    *   **截止时间**: 设置最后提交期限。
3.  点击 **“立即发布”**，学生将立即收到 QQ 推送。

### 2.3 批改与成绩管理
1.  **查看提交**: 在作业列表中点击 **“批改”**，查看全班提交状态（已交/未交）。
2.  **AI 预评**: 系统会显示 AI 给出的参考分数和评语。
3.  **人工复核**:
    *   点击学生姓名进入详情页。
    *   查看学生提交的内容（文本或文件预览）。
    *   在右侧评分栏修改分数或评语。
    *   点击 **“确认成绩”**，系统将推送最终成绩给学生。
4.  **导出成绩**: 点击列表上方的 **“导出 Excel”** 按钮下载报表。

## 3. 学生操作指南

### 3.1 加入班级
1.  在 QQ 聊天框发送指令 `/join 邀请码`（例如 `/join AB12CD`）。
2.  机器人回复“成功加入 [2026春季-软件工程] 班级”。

### 3.2 接收作业通知
*   当老师发布新作业时，机器人会自动发送卡片消息：
    > 📢 **新作业发布**
    > 标题：第一次实验报告
    > 截止：2026-03-01 23:59
    > [点击查看详情]

### 3.3 提交作业
方式一：**Web 端提交**
1.  点击通知卡片中的链接进入作业页面。
2.  在文本框输入内容，或点击上传按钮上传文件。
3.  点击 **“提交”** 按钮。

方式二：**QQ 快捷提交**
1.  向机器人发送 `/submit 作业ID`。
2.  根据提示直接发送文本或文件。
3.  机器人回复“接收成功，AI 正在评分中...”。

### 3.4 查看成绩
*   评分完成后，机器人会推送成绩单：
    > ✅ **作业已评分**
    > 标题：第一次实验报告
    > 分数：**85** / 100
    > AI 评语：代码结构清晰，但缺少必要的注释...
    > [查看详细反馈]

## 4. 常见问题 (FAQ)

**Q: 提交错了可以重交吗？**
A: 在截止日期前，您可以多次提交，系统将以最后一次提交为准。

**Q: 为什么我收不到通知？**
A: 请检查是否屏蔽了机器人消息，或确认是否已成功绑定身份。

**Q: 文件上传失败怎么办？**
A: 单个文件限制在 50MB 以内，支持 PDF, DOCX, JPG, PNG, ZIP 格式。建议在网络良好的环境下重试。
