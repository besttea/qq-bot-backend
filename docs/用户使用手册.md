# 用户使用手册 (User Manual)

**版本**: 1.1
**日期**: 2026-02-06

---

## 1. 简介

欢迎使用 **QQ 智能教学助手**！本系统是一款基于 QQ 机器人的智能化教学管理工具，旨在帮助老师高效发布与批改作业，帮助学生便捷提交作业并获得即时 AI 反馈。

### 核心功能
*   🤖 **QQ 交互**: 无需下载 App，在 QQ 中即可完成作业接收与提交。
*   ⚡ **AI 秒评**: 提交作业后，AI 助教将即时分析并给出参考评分与建议。
*   📊 **数据看板**: 教师可随时查看班级提交概况与成绩分布。

## 2. 快速开始 (Quick Start)

### 2.1 关注机器人
在使用服务前，请先添加我们的 QQ 机器人为好友，或将其邀请至您的班级群中。
*   **机器人 QQ 号**: `123456789` (示例)

### 2.2 身份绑定 (首次使用必做)
为了确保数据安全，您需要将 QQ 账号与真实的教务身份绑定。

**操作步骤**:
1.  与机器人发起私聊。
2.  发送绑定指令：
    ```text
    /bind <姓名> <学号/工号>
    ```
    *示例*: `/bind 张三 2023001`
3.  等待机器人回复“✅ 绑定成功！”，即完成注册。

> **注意**: 如果您是教师，绑定后系统会自动识别您的教师权限。

## 3. 学生使用指南

### 3.1 加入班级
老师会提供一个 **6位邀请码**，请通过以下方式加入：

**指令**:
```text
/join <邀请码>
```
*示例*: `/join AB12CD`

**反馈**:
> 🎉 成功加入 **[2026春季-软件工程]** 班级！
> 当前任课老师：李教授

### 3.2 查看作业
您可以随时查询当前待完成的作业。

**指令**:
```text
/todo
```

**反馈**:
> 📋 **待办作业列表**
> 1. **[未提交]** 第一次实验报告 (截止: 03-01 23:59) - ID: `101`
> 2. **[未提交]** 期中复习题 (截止: 03-05 12:00) - ID: `102`
>
> 发送 `/submit <ID>` 开始提交。

### 3.3 提交作业
支持 **文本直接提交** 和 **文件上传** 两种方式。

#### 方式一：文本提交 (适用于简答题、代码片段)
**指令**:
```text
/submit <作业ID> <内容>
```
*示例*: `/submit 101 本次实验通过冒泡排序实现了...`

#### 方式二：文件提交 (适用于文档、图片、压缩包)
1.  发送指令 `/submit <作业ID>` (不带内容)。
2.  机器人会提示：“请在 3 分钟内发送文件”。
3.  直接发送文件（支持 PDF, Word, JPG, ZIP）。
4.  机器人接收后回复：“📄 文件接收成功，AI 正在评分中...”。

### 3.4 查看成绩与反馈
作业批改完成后（AI 自动批改或老师复核），机器人会主动推送成绩单。您也可以主动查询。

**指令**:
```text
/score <作业ID>
```

**反馈**:
> ✅ **作业成绩单**
> *   **作业**: 第一次实验报告
> *   **得分**: **85** / 100
> *   **评语**: 代码逻辑清晰，但在边界条件处理上略有欠缺，建议参考...

## 4. 教师使用指南

教师主要通过 **Web 管理后台** 进行操作，部分快捷功能可通过 QQ 指令完成。

*   **管理后台地址**: `http://localhost:3000` (测试环境) / `https://admin.example.com` (正式环境)

### 4.1 创建班级 (Web 端)
1.  登录管理后台，进入 **“班级管理”**。
2.  点击 **“新建班级”**。
3.  输入班级名称（如“2026春季-计算机网络”）。
4.  系统生成 **邀请码** (如 `XY99ZZ`)，请复制并发送给学生。

### 4.2 发布作业 (Web 端)
1.  进入指定班级，点击 **“发布作业”**。
2.  填写详情：
    *   **标题**: 必填。
    *   **描述**: 支持 Markdown 格式。
    *   **附件**: 可上传课件。
    *   **截止时间**: 必填。
3.  点击发布，所有学生将立即收到 QQ 推送。

### 4.3 快捷发布 (QQ 端)
对于简单的通知类作业，可直接通过 QQ 发布。

**指令**:
```text
/publish <班级邀请码> <标题> <截止日期>
```
*示例*: `/publish XY99ZZ 阅读第三章 2026-03-10`

### 4.4 批改作业 (Web 端)
1.  进入 **“作业列表”**，点击 **“批改”**。
2.  系统列表展示所有学生的提交状态及 AI 预评分数。
3.  **人工复核**:
    *   点击学生姓名，预览作业内容。
    *   修改 AI 给出的分数（如果觉得不准确）。
    *   添加人工评语。
4.  点击 **“确认”**，成绩生效并推送给学生。

## 5. 常见问题 (FAQ)

**Q1: 绑定时提示“学号不存在”怎么办？**
> 请联系管理员确认您的学号是否已录入系统白名单。

**Q2: 作业提交后可以修改吗？**
> 在截止时间之前，您可以重复执行提交指令，系统将以最后一次提交的内容为准。截止时间后将无法提交。

**Q3: 支持哪些格式的文件？**
> 目前支持 `.pdf`, `.doc`, `.docx`, `.jpg`, `.png`, `.zip`。单个文件大小请控制在 50MB 以内。

**Q4: AI 评分不准确怎么办？**
> AI 评分仅供参考。所有成绩最终都会经过老师的人工复核，如果您对成绩有异议，请直接私聊老师或在课后反馈。
